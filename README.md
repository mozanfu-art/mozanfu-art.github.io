# Portfolio

## Overview
This repository contains the source code for my **personal portfolio website**, designed to showcase my IT projects, web development skills, and professional experience. The website is fully responsive, modern, and recruiter-friendly, built to highlight projects, skills, and contact information in an engaging format.

## Features
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices using Tailwind CSS.
- **Projects Showcase**: Interactive cards displaying live demos, GitHub repositories, and project details.
- **Technology Stack**: Built with **React**, **Tailwind CSS**, and **JavaScript**, leveraging modern frontend development practices.
- **Navigation & UI**: Smooth scrolling, hover animations, and clear sections for Skills, Projects, and Contact.
- **Extensible & Modular**: Components are reusable and easily updated for new projects or sections.

## Live Demo
Visit the live website here: [https://mozanfu-art.github.io](https://mozanfu-art.github.io)

## Technologies Used
- **Frontend**: React, Tailwind CSS, JavaScript
- **Icons & Animations**: lucide-react, motion/react
- **Deployment**: GitHub Pages

## Usage

1. Clone the repository:
git clone https://github.com/mozanfu-art/Portfolio.git
2. Install dependencies:
npm install
3. Start the development server:
npm start
4. Build for production:
npm run build

## Purpose

This portfolio is designed to showcase practical IT, web development, and project management skills. It serves as a professional introduction to recruiters, highlighting real-world projects, interactive demos, and structured documentation.
